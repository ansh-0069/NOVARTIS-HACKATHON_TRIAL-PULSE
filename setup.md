## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/ansh-0069/NOVARTIS-HACKATHON_TRIAL-PULSE
cd NOVARTIS-HACKATHON_TRIAL-PULSE
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```
http://localhost:5000
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

---

### 4️⃣ Python Scientific Engine

```bash
pip install xlsxwriter
python app.py
```
